Git configuration:

git config --global user.name "Yourname"
git config --global user.email <email-id>
git config --global core.editor nano

Git Commands:

Create folder
git init
create a file
git status
What is staging area?
git add.
git status
git commit/ git commit -m "Your message"
git status
git log --oneline = see a brief log of commits
git checkout <commit> <file>
git reset HEAD index.html = undo local changes to the state
git checkout --<file> = revert back to initial state
